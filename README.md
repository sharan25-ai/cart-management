# ShopKart - React Shopping Cart Application

ShopKart is a modern and responsive shopping cart web application developed using React and TypeScript. The project is designed to provide a simple, smooth, and interactive online shopping experience where users can browse available products, add items to their cart, manage product quantities, apply coupon codes, and view the final payable amount with cashback rewards.

The application includes a product listing page that displays items such as laptops, headphones, keyboards, and mouse products. Each product is shown with its name, price, short description, and visual icon, making the interface clean and easy to use. Users can add products to the cart with a single click and increase or decrease the quantity of each item using plus and minus buttons.

One of the main highlights of this project is the dynamic cart system. The cart automatically updates the total number of items, subtotal, cashback amount, coupon discount, and final payable amount in real time. It also includes an empty cart message, product removal option, and a clear order summary section.

ShopKart also includes a coupon code feature where users can apply valid coupon codes such as SAVE10 and SAVE20 to receive discounts. If the user enters an invalid coupon code, the application displays an error message. In addition, the cashback reward system provides different cashback percentages based on the cart value, making the application more engaging and similar to real e-commerce platforms.

The user interface is designed with a clean layout, modern typography, rounded product cards, a sticky header, cart counter, attractive buttons, and responsive styling. The project demonstrates important frontend development concepts such as React state management, conditional rendering, event handling, dynamic calculations, reusable functions, and responsive UI design.

This project is suitable for beginners who want to understand how shopping cart functionality works in real-world e-commerce applications. It can also be extended further by adding features such as user authentication, product search, product filtering, backend database integration, order history, payment gateway support, and admin product management.

Overall, ShopKart is a beginner-friendly yet professional React project that showcases practical frontend development skills and provides a strong foundation for building a complete e-commerce shopping application.
